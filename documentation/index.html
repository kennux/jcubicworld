Setting up Eclipse

Hello and welcome to the cubicworld documentation.
In this first step we will prepare eclipse for plugin programming.
First, you have to install the gradle plugin.
In order to do so, you must open the "Install Software" dialog which is located under Help > Install new software.
Click on the "Add" Button and add the gradle repository url for your Eclipse version (Luna: http://dist.springsource.com/snapshot/TOOLS/gradle/nightly).

After you added the repository, install the Gradle IDE component:

Now your IDE is ready for the Gradle build system.

Now, checkout the cubicworld repository and import the project as gradle project (File > Import):

Press the "Build Model" button and finish the import.
Now you're ready for programming your plugin.
